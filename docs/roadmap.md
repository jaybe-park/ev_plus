# 로드맵

현재 플레이 가능한 상태이며, 아래 항목들이 미구현으로 남아있다.

---

## 버그 / 미완성 로직

### 사이드팟 미구현
- **현상**: 올인 금액이 다른 여러 명이 동시에 올인하면 팟 분배가 부정확함
- **영향**: 3명 이상이 서로 다른 스택으로 올인할 때만 발생. 1:1 또는 동일 스택 올인은 정상
- **우선순위**: Medium

### 헤즈업 프리플랍 베팅 순서
- 표준 룰에서 헤즈업은 BTN/SB가 프리플랍에 먼저 행동
- 현재 코드가 이를 정확히 처리하는지 추가 검증 필요
- **우선순위**: Low (2인 게임을 주로 하지 않으면 영향 없음)

---

## 미연결 기능

### DB 기록 (`db/`)
- 스키마와 `recorder.py`는 완성됐지만 게임 세션과 연결이 안 됨
- 연결하면 핸드별 액션 기록, GTO 빈도 비교 분석 가능
- **작업 범위**: `server/session.py`의 `_apply()` 메서드에 기록 호출 추가

---

## 기능 추가 후보

### GTO 데이터 확장
- 현재: 프리플랍 일부 상황만 있음
- 추가 가능: vs_3bet 레인지, SB vs 각 포지션, 포스트플랍 레인지

### 포스트플랍 GTO 어드바이저
- 현재 포스트플랍은 봇과 힌트 모두 휴리스틱
- 보드 텍스처(드라이/웻), 포지션, 팟 오즈를 고려한 기본 레인지 추가 가능

### 통계 화면
- DB 연결 후: 핸드 히스토리, VPIP/PFR, GTO 대비 실제 플레이 빈도 비교

### UI 개선
- 애니메이션 (카드 딜, 칩 이동)
- 모바일 레이아웃 최적화
- 핸드 히스토리 패널
